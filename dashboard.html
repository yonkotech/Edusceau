Bienvenue a vous.

Ce site a été réalisé dans le cadre d’un travail pratique visant à simuler un système de vérification de bulletins scolaires. Il s’inspire du fonctionnement d’une plateforme officielle comme celle du Ministère de l’EPST (RDC), sans en être une version authentique ni connectée à une base de données réelle.

L’objectif principal de ce projet est de mettre en pratique nos compétences en HTML, CSS et JavaScript. Il s'agit de créer une interface utilisateur claire et fonctionnelle qui permettrait, dans un cas réel, de vérifier l’authenticité d’un document à l’aide d’un identifiant ou d’un code unique.

Ce site est donc une démonstration de ce que pourrait être un outil de vérification numérique dans le secteur éducatif. Il ne traite aucune donnée réelle, mais sert uniquement à illustrer les étapes d’un projet web complet : conception, structuration, design et interaction de base.
